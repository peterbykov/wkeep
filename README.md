####WKEEP
===========================================

Wkeep is analog [Google keep](https://www.google.com/keep/)

You need to install:
    1. [Composer](https://getcomposer.org/)
    2. [Npm](https://www.npmjs.com/)
    3. [Bower](http://bower.io/)
Installation:
    1. clone project
    2. run composer update
    3. create database "keep"
    4. doctrine migrate
    5. gulp concat
